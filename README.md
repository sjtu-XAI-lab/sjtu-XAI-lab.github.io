## 如何修改/添加内容
- 主要的文件是index.html。如果需要修改内容，直接修改index.html即可。
- 关于照片，放在img下面，最好是1：1的照片不然会失真。
- 关于publication：需要大家提供每一个publication一张代表性的图片
- 关于blog：直接在blog后面添加即可。
